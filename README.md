# Credit Card Fraud Detection using Machine Learning
##  Project Overview
This project aims to detect fraudulent credit card transactions using Machine Learning techniques.
The dataset is highly imbalanced, where fraudulent transactions are very rare compared to normal transactions.
A Logistic Regression model is trained to classify whether a transaction is **Fraud** or **Legitimate**.


## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Dataset
The dataset contains anonymized credit card transactions made by European cardholders.
It includes features extracted using PCA transformation and a target variable:
* **0 → Legitimate Transaction**
* **1 → Fraudulent Transaction**

## Model Used
Logistic Regression classifier is used to train the model on the dataset.
Steps followed:
1. Data Loading
2. Data Preprocessing
3. Handling Imbalanced Dataset
4. Train-Test Split
5. Model Training
6. Model Evaluation

##  Results
The model successfully identifies fraudulent transactions.
Evaluation metrics used:
* Accuracy
* Precision
* Recall
* F1 Score

##  How to Run
1. Download the dataset
2. Open the Jupyter Notebook
3. Run all the cells step-by-step
